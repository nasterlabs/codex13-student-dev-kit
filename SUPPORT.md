# Support

General questions:

- GitHub Discussions are enabled and are the preferred place for general
  questions.
- Email: `support@naster.dev`

Bug reports:

- GitHub Issues

Feature requests:

- GitHub Issues or Discussions

Security vulnerabilities:

- Do not open public issues.
- Use GitHub Private Vulnerability Reporting or email `security@naster.dev`.

Links:

- Issues: https://github.com/nasterlabs/codex13-student-dev-kit/issues
- Discussions: https://github.com/nasterlabs/codex13-student-dev-kit/discussions
- Product page: https://codex13.dev/student-dev-kit

Generated installers may download large external archives and modify the chosen
installation directory. Include install logs when reporting installer issues,
after removing personal paths or secrets.
