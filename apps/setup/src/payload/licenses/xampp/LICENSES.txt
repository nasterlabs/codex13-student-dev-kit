XAMPP for Windows - License Information
========================================

Codex 13 Student Dev Kit downloads the official Apache Friends portable package:

  xampp-portable-windows-x64-8.2.12-0-VS16.zip

Source URL configured in Setup:

  https://downloads.sourceforge.net/project/xampp/XAMPP%20Windows/8.2.12/xampp-portable-windows-x64-8.2.12-0-VS16.zip

XAMPP is a free, open source, cross-platform web server package developed and
distributed by Apache Friends (https://www.apachefriends.org/).

Apache Friends publishes the XAMPP compilation under the terms of the GNU
General Public Licence. XAMPP also bundles many independent open source
components, and each bundled component remains governed by its own license
terms. The 8.2.12 Windows portable archive includes a full upstream license
tree at:

  xampp\licenses\

After installation by Codex 13 Student Dev Kit, that upstream license tree is
available under:

  tools\xampp\licenses\

Important included top-level component license files include:

  tools\xampp\apache\LICENSE.txt
  tools\xampp\apache\NOTICE.txt
  tools\xampp\php\license.txt
  tools\xampp\phpMyAdmin\LICENSE
  tools\xampp\tomcat\LICENSE
  tools\xampp\tomcat\NOTICE

XAMPP is downloaded and installed by the Codex 13 Student Dev Kit installer
from its official source. The Codex 13 Student Dev Kit does not modify the
XAMPP binaries; only the XAMPP path-configuration scripts are patched for
portable operation. See the Codex 13 SDK repository for details on those
patches.

Codex 13 XAMPP patch scripts
----------------------------

The installer overlays a small source patch set onto XAMPP's path-configuration
scripts. The patch set is maintained in the Codex 13 Student Dev Kit source
repository under:

  apps/setup/src/patches/xampp/

The upstream-derived `install.php` patch and accompanying repair, verification
and diagnostic helpers are distributed as GPL-2.0-only. The full corresponding
source is in the repository, and installer builds also install those PHP files as
source text under:

  tools\xampp\install\

The repository license notice for the patch set is:

  LICENSES/LicenseRef-xampp-patch.txt

GPL-2.0-only text:

  LICENSES/GPL-2.0-only.txt
