# Security Policy

## Reporting A Vulnerability

Please do not open public issues for vulnerabilities.

Use GitHub Private Vulnerability Reporting:

```text
https://github.com/nasterlabs/codex13-student-dev-kit/security/advisories/new
```

or email:

```text
security@naster.dev
```

Include:

- affected version or commit,
- reproduction steps,
- expected and actual impact,
- any relevant logs or screenshots with personal data removed.

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.7.0-alpha.* | Yes, current public alpha line |
| Older snapshots or local builds | No |

After the first stable release the table above will list each release branch and
its support status. Only the latest patch release of each supported minor version
receives security fixes. Older release branches will be marked as unsupported here
when they reach end of life.
